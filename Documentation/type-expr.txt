* Type expressions

Type expressions are used a) as object literals b) for object destructuring.
See syntax.txt for type expression sugar.

(type name-or-null &rest slot-specs)

slot-specs ::= list of (.slot-name slot-value)

(<movie> loulou ((.name "Loulou") (.year "1981")))

A parameter is said to be typed if it is a type expression or type name.
